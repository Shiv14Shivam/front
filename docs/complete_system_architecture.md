- **Customers**: Browse vendor listings → cart → order → pay now/pay-later

- **Key Feature**: Flexible payment - immediate Razorpay OR credit (vendor approval, 1-7 days)

### 💳 Payment Flow (Sophisticated)
```
OrderItem states:
pending → vendor accept → accepted/unpaid
  ├─ pay-now → Razorpay verify → processing/paid → stock deduct → delivered/paid
  └─ pay-later → accepted/pay_later → vendor approve → processing/pay_later → customer pays by due → delivered/paid
      └─ vendor reject → declined/unpaid
```
